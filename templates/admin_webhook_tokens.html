About Webhook Authentication

Each token creates a unique webhook URL in the format:

https://yourserver.com/webhook/{TOKEN}

Use these URLs in your TradingView alert webhooks. The token in the URL provides authentication and ensures only authorized services can send webhook data to your system.

Important:
  • Keep your webhook tokens secure. Anyone with the token can send data to your system.
  • If you suspect a token has been compromised, delete it and create a new one.
  • Each integration (like TradingView) should have its own token for better security and tracking.
